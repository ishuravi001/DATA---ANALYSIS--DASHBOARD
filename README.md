# CALL CENTRE DATA ANALYSIS (Interactive Dashboard using MS Excel)
## Project Objective
The objective of this project is to create an interactive call centre dashboard that summarizes key performance metrics such as calls, revenue, ratings, customers, and call duration. The dashboard helps track trends, compare agent and regional performance, and support quick, data-driven decision-making to improve efficiency and customer satisfaction.
